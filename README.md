# Clarity MCP Server - Railway Deployment

## Quick Deploy

1. Create new Railway project
2. Connect this repository
3. Add environment variables:
   - DB_SERVER
   - DB_USER
   - DB_PASSWORD
   - DB_NAME
   - PORT
4. Deploy!

## Environment Variables
See .env.example for required variables.
